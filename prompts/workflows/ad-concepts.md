# Ad Concepts Prompt

Create at least 5 Meta ad concepts.

Use:

- ICP brief
- competitor analysis
- offer notes
- brand voice

Each concept must include:

- concept name
- target segment
- core pain
- hook
- visual direction
- primary text
- 3 headline variants
- 3 primary text variants
- CTA
- landing page alignment
- why it might work
- risk / caveat
