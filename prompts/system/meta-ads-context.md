# Meta Ads Context

This repo focuses on Meta ads for B2B SaaS and service businesses.

Useful creative patterns:

- problem/solution
- founder POV
- before/after
- painful status quo
- demo/use case
- objection handling
- competitor alternative
- cost of inaction
- proof-led if proof exists

Remember:

- Meta is interruption-based.
- The hook has to earn attention quickly.
- Visual direction matters as much as copy.
- Landing page message match is part of the ad concept.
- Most B2B ads fail by sounding too abstract.
