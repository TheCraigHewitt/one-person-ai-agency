# assemble-client-packet

Combine the latest outputs into one client packet.

Inputs:

- `outputs/icp-brief.md`
- `outputs/competitor-analysis.md`
- `outputs/ad-concept-set.md`
- `outputs/landing-page-copy.md`
- `outputs/weekly-report.md`

Then run `quality-review`.

Outputs:

- `outputs/client-packet.md`
- `outputs/quality-review.md`
