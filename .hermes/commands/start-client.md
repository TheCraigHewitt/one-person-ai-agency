# start-client

Create a new client workspace from templates.

Ask Craig for:

- client slug
- website URL
- offer
- primary audience
- known competitors

Then create:

```text
clients/<slug>/inputs/
clients/<slug>/outputs/
```

Copy the input templates from `templates/inputs/`.

After setup, run the `client-intake` skill and write `clients/<slug>/outputs/intake-summary.md`.
