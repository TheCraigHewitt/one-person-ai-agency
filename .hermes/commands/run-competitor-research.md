# run-competitor-research

Run competitor research for a client.

Priority:

1. Use Foreplay if `FOREPLAY_API_KEY` is available.
2. Use public competitor websites.
3. Use `ad-example-library.md` fallback.

Outputs:

- `clients/<client>/inputs/ad-example-library.md`
- `clients/<client>/outputs/competitor-analysis.md`

Do not invent performance data.
