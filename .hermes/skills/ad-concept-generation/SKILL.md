# Ad Concept Generation

Use this skill to generate usable Meta ad concepts from ICP and competitor research.

## Inputs

- `clients/<client>/outputs/icp-brief.md`
- `clients/<client>/outputs/competitor-analysis.md`
- `clients/<client>/inputs/offer-notes.md`
- `clients/<client>/inputs/brand-voice.md`

## Output

Write `clients/<client>/outputs/ad-concept-set.md`.

## Requirements

Create at least 5 concepts.

Each concept must include:

- concept name
- target segment
- core pain
- hook
- visual direction
- primary text
- 3 headline variants
- 3 primary text variants
- CTA
- landing page alignment
- why it might work
- risk / caveat

## Rules

- Make concepts distinct from each other.
- Use Foreplay patterns as input, not as copy-paste material.
- Do not guarantee results.
