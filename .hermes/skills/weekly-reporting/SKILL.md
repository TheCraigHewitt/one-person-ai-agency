# Weekly Reporting

Use this skill to turn campaign metrics into a weekly client report draft.

## Inputs

- `clients/<client>/inputs/campaign-metrics.csv`
- campaign context
- prior recommendations if available

## Output

Write `clients/<client>/outputs/weekly-report.md`.

## Required Sections

- Executive summary
- What changed
- Winners
- Losers
- Hypotheses
- Recommended next actions
- Questions for client

## Rules

- Label sample data.
- No fake causality.
- Separate facts from interpretation.
- Do not imply the system managed the ad account.
