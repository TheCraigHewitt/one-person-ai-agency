# Landing Page Copy

Use this skill to draft landing page and ad copy for a selected concept.

## Inputs

- selected ad concept
- `clients/<client>/outputs/icp-brief.md`
- `clients/<client>/inputs/offer-notes.md`
- `clients/<client>/inputs/brand-voice.md`

## Output

Write `clients/<client>/outputs/landing-page-copy.md`.

## Required Sections

- Hero
- Problem framing
- Mechanism
- Proof placeholders
- Offer section
- Objection handling
- CTA section
- Meta ad copy variants

## Rules

- Use proof placeholders if proof is missing.
- Do not invent testimonials, customer names, metrics, or logos.
- Keep copy practical and specific.
