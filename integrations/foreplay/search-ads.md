# Search Ads

Use this when Hermes needs broad creative examples.

Inputs:

- client ICP
- category keywords
- pain point keywords
- competitor brands
- domains

Search strategy:

1. Search exact competitor brand names.
2. Search competitor domains.
3. Search category keywords.
4. Search pain point keywords.
5. Prefer active Meta ads where possible.

Output:

Normalize all usable results into `ad-example-library.md`.
