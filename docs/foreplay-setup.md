# Foreplay Setup

Foreplay is the V1 ad research provider.

1. Start the Foreplay free trial.
2. Create an API key.
3. Add it to `.env`.

```text
FOREPLAY_API_KEY=
```

For the video, pre-run Castos competitor research and save normalized examples to:

```text
examples/castos/inputs/ad-example-library.md
```

Then show one live refresh only if auth is stable.
