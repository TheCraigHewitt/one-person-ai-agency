# Quickstart

1. Copy `.env.example` to `.env`.
2. Add your Foreplay API key.
3. Review `.hermes.md`.
4. Open the Castos example.
5. Ask Hermes to run the ICP workflow.

Example:

```text
Run the ICP research workflow for examples/castos.
```

Then:

```text
Use the Castos competitor inputs and Foreplay ad examples to generate 5 Meta ad concepts.
```
